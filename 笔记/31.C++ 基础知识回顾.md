# C++ 基础知识回顾

1. ### **作用域与变量生命周期**

```C++
#include <iostream>
using namespace std;

int main() {
    int x = 10;
    {
        int x = 20;
        cout << "Inner x: " << x << endl;
    }
    cout << "Outer x: " << x << endl;
    return 0;
}
```
问题：输出是什么？为什么？

2. ### **函数重载与类型转换**

```c++
#include <iostream>
using namespace std;

void func(int x) {
    cout << "int version" << endl;
}

void func(double x) {
    cout << "double version" << endl;
}

int main() {
    func(5);
    func(5.0);
    func('a');
    return 0;
}
```
问题：输出是什么？为什么？

3. ### 指针与引用的区别

```c++
#include <iostream>
using namespace std;

void modify(int* ptr, int& ref) {
    *ptr = 10;
    ref = 20;
}

int main() {
    int a = 5, b = 5;
    modify(&a, b);
    cout << "a: " << a << ", b: " << b << endl;
    return 0;
}
```
问题：输出是什么？为什么？

6. ### 虚函数与静态绑定

```c++
#include <iostream>
using namespace std;

class Base {
public:
    virtual void show() { cout << "Base" << endl; }
};

class Derived : public Base {
public:
    void show() { cout << "Derived" << endl; }
};

int main() {
    Base b;
    Derived d;
    Base* ptr = &d;
    ptr->show();
    return 0;
}
```
问题：输出是什么？为什么？

9. ### 右值引用与移动语义

```c++
#include <iostream>
using namespace std;

void func(int&& x) {
    cout << "Rvalue reference: " << x << endl;
}

int main() {
    int a = 5;
    func(a);
    func(10);
    return 0;
}
```
